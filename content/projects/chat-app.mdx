## Description

Flamingo Chat is a chat application featuring a sleek interface. Users can sign
in with GitHub or Discord and join chat groups. The interface includes sections
for Home, Emotes, Groups, and Users, with real-time messaging capabilities.
Users can share text and images, and interact in group chats.
